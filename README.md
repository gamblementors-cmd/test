## Contributing

All work is tracked via Paperclip. Engineers pick up assigned tasks, implement them, and mark done.

Commit convention:
- `feat:` new feature
- `fix:` bug fix
- `chore:` maintenance

Always add `Co-Authored-By: Paperclip <noreply@paperclip.ing>` to commits.
